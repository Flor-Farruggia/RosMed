Sobre el desarrollo del producto:

Este producto esta siendo desarrollado por Farruggia, Florencia. Se utilizaron las herramientas de desarrollo, mediante el uso de un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web, el cual se llama Visual Studio Code (VSC). En el mismo se desarrolló el código en formato HTML para darle estructura al documento y por último para dar estilo se utilizó CSS.

Algunas consideraciones: Actualmente esta aplicación está en plena producción, el estado de la misma es solo de previsualización, la página solo dispone de la opción de verla en sus diferentes tamaños al ser responsive, lo que implica que podremos ver como la aplicación se verá en los diferentes dispositivos, es decir, el usuario no será capaz de interactuar con la página.
