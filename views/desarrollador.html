Sobre el desarrollo del producto:

Este producto esta siendo desarrollado por Farruggia, Florencia. Se utilizaron las herramientas de desarrollo, mediante el uso de un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web, el cual se llama Visual Studio Code (VSC). En el mismo se desarrolló el código en formato HTML para darle estructura al documento, para dar estilo se utilizó CSS y por último para crear una mínima interacción se usó JavaScript.

Algunas consideraciones: Actualmente esta aplicación está en plena producción, el estado de la misma es solo de previsualización, la página solo dispone de la opción de verla en sus diferentes tamaños al ser responsive, lo que implica que podremos ver como la aplicación se verá en los diferentes dispositivos. Por lo tanto, la página no cuenta con una Base de Datos o con herramientas de Back-end.
